A social network app, Socially where user can post, follow friends and family, like and dislike the posts, watch others profiles and more made with Django using MongoDB as database.

To run the code use the following commands:
1. pip install -r requirements.txt

2. pip install djongo

3. python3 manage.py makemigrations

4. pyhton3 manage.py migrate

5. python3 manage.py runserver
